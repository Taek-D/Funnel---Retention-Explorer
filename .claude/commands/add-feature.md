## 작업 순서

1. 요청된 기능을 분석하고 영향받는 파일 파악:
   - UI/레이아웃 → index.html, styles.css
   - 데이터 처리/분석 로직 → app.js (AppState 확인)
   - 차트/시각화 → charts.js
2. 기존 코드 패턴을 먼저 읽고 동일한 패턴으로 구현
3. 구현 완료 후 체크리스트 확인
